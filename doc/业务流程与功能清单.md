## 一、用户注册与认证流程

### 1.1 客户注册流程

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│ 访问入口 │───►│ 选择注册 │───►│ 验证手机 │───►│ 完善信息 │
│         │    │  方式   │    │         │    │         │
└─────────┘    └────┬────┘    └─────────┘    └────┬────┘
                    │                              │
              ┌─────┴─────┐                  ┌─────┴─────┐
              │ 快捷注册   │                  │ 正式客户  │
              │ (临时客户) │                  │ 档案归档  │
              └───────────┘                  └───────────┘
```

**对应功能**:

| 功能点 | 功能描述 |
|--------|----------|
| 快捷注册链接生成 | 客服/主管生成带有效期的注册链接 |
| 手机验证码注册 | 输入手机号获取验证码完成注册 |
| 临时客户创建 | 自动生成临时客户档案，可立即咨询 |
| 客户信息完善 | 补充姓名、单位、职务等信息 |
| 档案归档 | 临时客户转正，关联正式机构档案 |

### 1.2 用户登录流程

```
┌─────────┐    ┌─────────┐    ┌─────────┐
│ 访问系统 │───►│ 选择登录 │───►│ 身份验证 │
│         │    │  方式   │    │         │
└─────────┘    └────┬────┘    └────┬────┘
                    │              │
              ┌─────┴─────┐  ┌─────┴─────┐
              │ 账号密码  │  │ 验证通过  │
              │ 手机验证码│  │ 进入系统  │
              │ 微信扫码  │  │           │
              └───────────┘  └───────────┘
```

**对应功能**:

| 功能点 | 功能描述 |
|--------|----------|
| 账号密码登录 | 用户名+密码登录 |
| 手机验证码登录 | 手机号+短信验证码登录 |
| 微信登录 | 微信扫码授权登录 |
| 密码找回 | 通过手机验证码重置密码 |
| 强制修改密码 | 首次登录强制修改初始密码 |

